Granula Project

A fine-grained performance evaluation systems for Big Data platforms.

Introduction

July 2016 by Wing Lung Ngai

Granula is a fine-grained performance evaluation system for Big Data platforms. There are four main modules:

  • Performance modelling:
  • Performance monitoring:
  • Performance archiving:
  • Performance visualizing:

Demo

We have integrated into several large-scale graph processing systems. Examples:

Examples (alpha release)

  • Giraph: running pagerank on ldbc-1000, with 20 nodes
  • Giraph: running bfs on ldbc-1000, with 20 nodes

Examples (beta release)

  • Giraph: running community detection on datagen-1000, with 8 nodes
  • Powergraph: running local clustering coefficient on datagen-1000, with 4 nodes
  • Powergraph: running local clustering coefficient on graph500-25, with 16 nodes
  • GraphmatD: running pagerank on datagen-1000, with 1 nodes
